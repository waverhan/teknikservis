# Global Architecture Rules

1. Keep business logic in web SaaS.
2. Flutter app must NOT duplicate business logic.
3. All data must be scoped by business_id.
4. Use environment variables.
5. Use TypeScript everywhere (web).
6. Keep code modular and readable.
7. Prefer simplicity over abstraction.
8. Avoid over-engineering.
9. Maintain Railway compatibility.
10. Printing is always client-side.
